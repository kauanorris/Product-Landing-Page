# 🥤 Coca-Cola Landing Page (Clone)

Projeto de uma landing page inspirada no site da Coca-Cola, desenvolvido com foco em **HTML, CSS e responsividade**.

---

## 📌 Sobre o projeto

Este projeto foi criado com o objetivo de praticar:

- Estruturação semântica com HTML  
- Estilização avançada com CSS  
- Layouts com **Flexbox** e **Grid**  
- Responsividade com **Media Queries**  
- Organização visual moderna (estilo landing page)

---

## 🚀 Funcionalidades

- ✅ Header com navegação  
- ✅ Menu hambúrguer (mobile)  
- ✅ Banner principal com CTA  
- ✅ Seção “More Enjoy” com grid responsivo  
- ✅ Seção de produtos  
- ✅ Footer completo com links e redes sociais  
- ✅ Layout responsivo (desktop → mobile)  

---

## 📱 Responsividade

O projeto foi desenvolvido com abordagem responsiva, incluindo:

- Menu adaptável (desktop → hambúrguer)  
- Grid que vira coluna no mobile  
- Ajustes de tipografia e espaçamento  
- Reorganização com Flexbox e Grid  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  
- Media Queries  

---

## 🎨 Destaques técnicos

- Uso de `clip-path` para background dinâmico  
- Centralização com `position: absolute` + `transform`  
- Uso de `order` para reorganização no mobile  
- `object-fit: cover` para imagens responsivas
